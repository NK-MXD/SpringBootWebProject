课程详情

万科教育旗下全资子公司，提供以深度体验为主题的专业周末营、冬夏令营，让冲浪、航海、登山、滑雪等户外运动伴随孩子们健康成长、全面发展。

专注于自然教育、探索教育、艺术教育、公益服务四大板块，旨在鼓励青少年参加户外活动，建立健康、快乐、积极的生活方式。使用美国德州教育体系（TEKS）并采用学分制的考核方式。学生完成指定的课程并取得相应学分后，将会获得美国南方院校协会（SACS）认证的美国高中学历证书，为学生升学提供完善的保障。

学校提供多种必修和选修课，覆盖英语类、第二外语类、数学类、自然学科类、人文学科类及社会学科类共6个学科类别，同时也提供多达16门的AP课程。AP课程及考试始于1955年，由美国大学理事会主持，美国教育考试中心（ETS）组织出题和考试。AP课程是在高中阶段开设的具有大学水平的课程，经过50多年的发展，AP课程现已覆盖20个门类，37个学科，其中包括 微积分、统计学、物理、计算机、经济学、英语、历史、信息技术、音乐、艺术等，学生可以根据自己的实际能力和将来发展方向选读AP课程。开设AP课程，在美国被视为高中学校教学实力的象征，美国众多优秀高中学校普遍开设AP课程。

主营帆船培训和航海夏、冬令营，致力于青少年帆船运动的推广和航海文化的传播。是中国大陆唯一获得英国皇家游艇协会RYA认证的中心，并使用其系统的进阶培训课程，拥有国际认证教练团队。

船艇及安全设备均采用国际品牌，拥有完备的安全流程和设备，为学员提供全面、专业的保障。AP课程可以使高中学生提前接触到大学课程，避免了高中和大学初级阶段课程的重复。同时，AP课程是进入美国名校的最有力保障。目前，已有60多个国家的近3600所大学承认AP学分为其入学参考标准，其中包括哈佛、耶鲁、牛津、剑桥等世界顶尖大学。

万科教育旗下全资子公司，成立至今已成为国内顶尖的青少年潜水培训机构。拥有深圳、三亚、广州三大教学基地，全年开展周末潜水体验营，青少年冬夏令营，成人课程及环球潜水旅行等。

2016年初由万有引力独自研发的青少年防溺水课程，已在校园和社区内进行推广。让孩子更安全、更从容地探索海洋。以万科梅沙教育为起点，联合梅沙社区学校、户外学校、艺术学校、运动学校、梅沙中学、梅沙书院等多类型的教育机构， 创建高效的互联网工作平台、教育服务沟通与交易平台、教育社区，连接更多的营地教育 、社区教育机构、教师、家长、受教育者。在梅沙科技的大家庭里，我们是一群善于学习的年轻人， 在瞬息万变的互联网潮流中，不断挑战新高度，拥抱更好的自己；我们是充满着无限灵感和活力的极客，勇于创新，在追求极致的道路上， 从不停歇；我们更是怀揣着理想的互联网团队，因拥有共同的理想而不断坚持， 无论前路如何，始终并肩而行。公司按照BAT薪酬标准，团队成员不仅仅可以获得有竞争力的薪酬和长期激励，更重要的是和一群有意思的人一起成长，一起成就有价值的教育产品。公司还有在其它公司绝对见不到的丰富多彩的团建活动，我们的团建活动就是带领团队体验户外营地教育学习帆船，潜水，划艇...
